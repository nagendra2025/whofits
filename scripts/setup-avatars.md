# Avatar Setup Instructions

Since there's no free API that provides recognizable caricatures of famous people, you have two options:

## Option 1: Use Free Caricature Generator (Recommended)

1. **Visit Fotor AI Avatar Generator**: https://www.fotor.com/features/ai-avatar-generator/
2. For each person, upload a reference photo and generate a "Caricature" or "Cartoon" style avatar
3. Download and save to `public/avatars/[person-id].png`

## Option 2: Use Free Illustration Sources

Search these sites for "[Person Name] caricature":
- **Pixabay**: https://pixabay.com/images/search/caricature/ (Free, no attribution needed)
- **OpenClipart**: https://openclipart.org/ (Public domain)
- **Flaticon**: https://www.flaticon.com/search?word=caricature (Some free with attribution)
